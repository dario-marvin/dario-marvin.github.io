Research

Forecasting Ground-Level Ozone with Machine Learning

In this study, we developed a machine learning framework to forecast next-day peak ground-level ozone concentrations in southern Switzerland—a region with sparse monitoring infrastructure and complex terrain. Instead of relying on spatial models, we focused on feature selection and interpretability to build accurate, location-specific predictors.

We compared a custom genetic algorithm with Shapley value-based feature selection, the latter offering insights into variable importance and feature interactions. Our analysis showed that forecasted temperature and NO₂ levels were the most influential features, with ozone formation exhibiting a nonlinear (sigmoid-like) response to temperature—consistent with known atmospheric chemistry.

Among the models tested, LSBoost and NGBoost achieved the best performance, especially in forecasting extreme ozone events. Feature selection significantly improved both prediction accuracy and model efficiency.

This research highlights how interpretable, data-driven models can support effective air quality forecasting, even in data-sparse and topographically complex regions.

Projects

ESP8266 IoT Sensor Node

Home environment monitoring system that tracks temperature, humidity, and pressure in different rooms using ESP8266 boards and BME280 sensors. The system is designed to be lightweight, easy to set up, and accessible to anyone connected to the local Wi-Fi network.

Machine Learning for Badly Scanned Digit Recognition

A machine learning project focused on recognizing digits from badly scanned documents. I created the dataset myself by generating, printing, and scanning a sequence of digits, then extracting the digit images using custom image processing techniques. The project explores and compares several classification algorithms, with k-nearest neighbors achieving the best results (99.7% accuracy). The process covers the full pipeline from dataset creation to model evaluation.

Machine Learning for Badly Scanned Letter Recognition

This project extends the digit recognition work to the recognition of individual letters from badly scanned documents. I created a new dataset by generating, printing, and scanning sequences of letters, then extracting the letter images using custom image processing. The project evaluates several machine learning classifiers, achieving high accuracy despite the challenges of misaligned and noisy scans. This work demonstrates the feasibility of applying similar techniques to more complex character recognition tasks.

Mandelbrot Fractal Visualizer (Ubuntu Console)

A C++ script that visualizes Mandelbrot fractals directly in the Ubuntu console. Each character space in the terminal is mapped to a point in the complex plane and colored according to the fractal's iteration behavior, creating a striking ASCII-art representation of the Mandelbrot set.
